# 书循－最便捷的大学生二手书交易平台
- [主页ishuxun.cn](http://ishuxun.cn)
- 微信号 ishuxun

## Features
- 居于[LeanCloud](http://leancloud.cn) Bass服务搭建后端
- 微信端使用[ionic](http://ionicframework.com) 框架做移动优先的HTML5网页

## Installation
```
npm install
```
## Build
```
npm build
```
## Run
```
npm start
```
## Test
```
npm test
```
